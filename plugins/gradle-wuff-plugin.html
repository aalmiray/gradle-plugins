Summary

This plugin automates assembly of OSGi/Eclipse-RCP/Eclipse-IDE bundles and applications.

Website https://github.com/akhikhl/wuff
Author akhikhl
Version 0.0.1

Usage

buildscript {
  repositories { jcenter() }

  dependencies {
    classpath 'org.akhikhl.wuff:wuff-plugin:0.0.1'
  }
}

apply plugin: 'java'
apply plugin: 'eclipse-ide-app'

repositories {
  mavenLocal()
  jcenter()
}

You may also define a more specific project type with

  • apply plugin: 'eclipse-bundle'
  • apply plugin: 'eclipse-config'
  • apply plugin: 'eclipse-equinox-app'
  • apply plugin: 'eclipse-ide-app'
  • apply plugin: 'eclipse-ide-bundle'
  • apply plugin: 'eclipse-rcp-app'
  • apply plugin: 'osgi-bundle'
  • apply plugin: 'swt-app'
  • apply plugin: 'swt-lib'
