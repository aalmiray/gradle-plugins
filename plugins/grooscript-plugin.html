Summary

This plugin provides grooscript actions. Convert groovy files to javascript, use a daemon to make conversions or create a static web project to use it.

Website https://github.com/chiquitinxx/grooscript-gradle-plugin
Author Jorge Franco
Version 0.2

Usage

buildscript {
    repositories {
        mavenCentral()
    }
    dependencies {
        classpath group: 'org.grooscript:grooscript-gradle-plugin:0.2'
    }
}

apply plugin: 'grooscript'
