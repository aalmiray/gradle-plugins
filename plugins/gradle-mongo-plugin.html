Summary

The Gradle Mongo Plugin allows you to run a managed instance of Mongo from your gradle build. It is based on the Embedded MongoDb project by Flapdoodle OSS.

Website https://github.com/sourcemuse/GradleMongoPlugin
Author sourcemuse
Version 0.1.0

Usage

buildscript {
    repositories { jcenter() }
    dependencies {
        classpath 'com.sourcemuse.gradle.plugin:gradle-mongo-plugin:0.1.0'
    }
}

apply plugin: 'mongo'
