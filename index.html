Welcome

Gradle is the next generation build tool for the JVM. It has a booming plugin community, alas there's no official site (yet) where one can find information about all available plugins (other than those bundled with the standard distribution, see here). This site is an attempt to have a somewhat organized list of useful Gradle plugins that do not ship with the standard distribution.

Do you know of a plugin that's missing from this list? Drop us a line at @aalmiray or better yet, send us a Pull Request as outlined here.
